Basic Outline
#####################################################################
This project demonstrate a working example on how to use spring data Jpa + JWT + Mysql in a spring boot project.

This project make use of JWT token for authentication.

A filter has also been created to authenticate requests with correct JWT token before firing the web service.

In this project a "/register" web service is registers the user without any authentication. "/authenticate" web service is used to create JWT that will be used in further authentication.

All other URI other than register and authenticate will comfront JWT authentication.
