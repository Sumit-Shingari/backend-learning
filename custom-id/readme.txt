Basic Outline
#####################################################################
This project demonstrate a working example on how to creating a custom id using spring data Jpa along with Mysql in a spring boot project.

In this example we are creating a customized primary key with sequence 10, 20, 30 ....... and so on. 

The project is using lambok for generating getters, setter and performing constructor DI in controller and service files.
