Basic Outline
#####################################################################
This project demonstrate a working example on how to use spring data Jpa along with Mysql in a spring boot project.

It focuses mainly on hibernate one to many (aggregration) example in detail.
 
This project shows how we can provide some default data in our application at startup using ApplicationListener<ContextRefreshedEvent>.

we have additionally modifed the response making the use Json serializable and json ignore properties.
