Basic Outline
#####################################################################
This project demonstrate a working example on how to use spring data Jpa along with Mysql in a spring boot project.

It focuses mainly on hibernate many to many (aggregration) example in brief.
 
This project shows how we can provide some default data in our application at startup using ApplicationListener<ContextRefreshedEvent>.

we have additionally modifed the response making the use json ignore properties.

The project is using lambok for generating getters, setter and performing constructor DI in controller and service files.
